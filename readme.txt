Add a dynamic copyright date block to your WordPress site that automatically updates with the current year.

== Description ==

The Copyright Date Block is a simple yet powerful Gutenberg block that helps you maintain an up-to-date copyright notice on your website. With this block, you can:

* Display the current year automatically
* Optionally show a date range (e.g., 2020 - 2025)
* Customize text color to match your theme
* Adjust font size for perfect presentation
* No coding required - simple settings in the block editor

Perfect for footers and any other location where you need to display copyright information that stays current without manual updates.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/copyright-date-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. In the block editor, add the "Copyright Date Block" wherever you want to display your copyright notice.
4. Use the block settings panel to customize the display options.

== Frequently Asked Questions ==

= How do I show a date range instead of just the current year? =

1. Select the Copyright Date Block in your editor
2. In the block settings panel on the right, enable "Show Starting Year"
3. Enter your desired starting year
4. The block will automatically show the range from your starting year to the current year

= Will the year update automatically? =

Yes! The current year is generated dynamically, so it will always show the correct year without any manual updates needed.

= Can I change the color or size of the text? =

Yes! The block supports:
* Text color customization through the color settings panel
* Font size adjustments through the typography panel
